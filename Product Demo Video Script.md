## TECHNICAL NOTES FOR VIDEO PRODUCTION

### Visual Style
- Clean, modern screen recordings
- Use cursor highlights and smooth transitions
- Add subtle zoom-ins to emphasize key features
- Use consistent color scheme matching the brand
- Include progress bars to show video timeline

### Animation Notes
- Fade transitions between sections (0.5 seconds)
- Use callout boxes that appear with slide-in animation
- Highlight clickable elements before cursor moves to them
- Use subtle motion graphics for data visualizations

### Audio
- Professional voiceover (clear, confident, warm tone)
- Background music: subtle, corporate, not overpowering
- Sound effects for button clicks and transitions (optional)

### Pacing
- Speak at moderate pace (150-160 words per minute)
- Pause 1-2 seconds after each major point
- Allow time for viewers to absorb visual information
- Keep each section under 2 minutes

### Accessibility
- Include closed captions
- Use high contrast for on-screen text
- Ensure all text is readable at various screen sizes
- Provide audio descriptions for key visual elements

### Call-to-Action Overlays
- Display URL prominently in final 15 seconds
- Show contact information clearly
- Include QR code for easy mobile access
- Add social proof elements if available (customer logos, testimonials)
